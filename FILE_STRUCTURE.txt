Quick Command Reference
=======================

Setup & Status:
  python test_setup.py                    # Verify setup
  python agent7.py status                 # Check system status

Project Management:
  python agent7.py create-project "Name" "Description"
  python agent7.py list-projects

Task Management:
  python agent7.py create-task <project_id> --type <type> --title "Title" --desc "Description"
  python agent7.py list-tasks
  python agent7.py list-tasks --project <id>
  python agent7.py list-tasks --status pending
  python agent7.py task-details <task_id>

Execution:
  python agent7.py execute-task <task_id>
  python agent7.py execute-workflow <project_id>

Examples:
  python example_usage.py                 # Run complete demo

Development Workflow
====================

1. First Time Setup:
   pip install -r requirements.txt
   python test_setup.py

2. Start LM Studio:
   - Load a model
   - Start local server
   - Verify at http://localhost:1234

3. Create Your First Project:
   python agent7.py create-project "Test" "My first project"
   python agent7.py create-task 1 --type planning --title "Plan" --desc "Test planning"
   python agent7.py execute-task 1
   python agent7.py task-details 1

4. Explore:
   - Check example_usage.py for code examples
   - Read QUICKSTART.md for more commands
   - Read ARCHITECTURE.md to understand the system
