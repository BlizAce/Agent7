Development Workflow
====================

1. First Time Setup:
   pip install -r requirements.txt
   python test_setup.py

2. Start LM Studio:
   - Load a model
   - Start local server
   - Verify at http://localhost:1234

3. Create Your First Project:
   python agent7.py create-project "Test" "My first project"
   python agent7.py create-task 1 --type planning --title "Plan" --desc "Test planning"
   python agent7.py execute-task 1
   python agent7.py task-details 1

4. Explore:
   - Check example_usage.py for code examples
   - Read QUICKSTART.md for more commands
   - Read ARCHITECTURE.md to understand the system
